HTML5 Web Standard

HTML5 is the new web development standard. It incorporates HTML and XML functionality into a versatile standard for web page development. Many of the features available today are displayed on this page or are enabled when hovering over the highlight text or image. There are also links below that will take you to other pages in other styles with more features like animation.

The design for this web page includes a left-side menu with hyper-links to other areas on this page. These web pages also use CSS3 (Cascading Style Sheets) which is a language that describes:

  • the style of an HTML document

  • how the HTML elements should be displayed

  • transitions and animation (follow the link below)
